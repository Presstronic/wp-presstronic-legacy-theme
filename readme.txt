== Description ==
A lightweight, fast WordPress theme that recreates the 2015 Presstronic one-page layout (hero + about + services + contact) with modern, dependency-free code.

Theme demo: https://presstronic.com/

== Installation ==
1. In your admin panel, go to Appearance -> Themes and click the Add New button.
2. Click Upload Theme and choose the presstronic-legacy zip.
3. Activate the theme.

== Usage ==
Use the Customizer to edit the homepage copy, hero image, and contact details.

== Frequently Asked Questions ==
= Does this theme include a blog? =
Yes. The theme supports standard posts and a basic blog index.

= Can I customize colors and typography? =
The theme ships with a minimal, curated design. Customizer controls are focused on content.
