# WordPress.org Submission Checklist

## Required files
- [ ] `style.css` with correct header fields (Theme Name, Version, Text Domain)
- [ ] `index.php`, `comments.php`, `screenshot.png` (max 1200x900)
- [ ] `readme.txt` in WP.org format
- [ ] License file (`license.txt`)

## Theme checks
- [ ] Run Theme Check (locally) and address REQUIRED issues
- [ ] Import Theme Unit Test data and verify output
- [ ] Verify no PHP errors/notices

## i18n and text domain
- [ ] Ensure only one text domain: `presstronic-legacy`
- [ ] Confirm folder name matches slug: `presstronic-legacy`

## Assets and licensing
- [ ] Verify all images/fonts are properly licensed
- [ ] List third-party assets in `readme.txt`

## Submission prep
- [ ] Generate a clean zip (no dev files)
- [ ] Confirm stable tag in `readme.txt`
- [ ] Create a WordPress.org account (if needed)

## Submit
- [ ] Upload zip at https://wordpress.org/themes/upload/
- [ ] Respond to reviewer feedback
