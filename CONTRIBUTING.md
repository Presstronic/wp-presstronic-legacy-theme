# Contributing

Thanks for considering a contribution!

## Workflow
- Fork the repo and create a feature branch.
- Make changes with clear, focused commits.
- Run `composer install` and `vendor/bin/phpcs` locally.
- Open a PR with a concise description and screenshots if UI changed.

## Code style
- Follow WordPress Coding Standards.
- Use tabs for indentation in PHP templates.
- Keep changes minimal and documented.

## Reporting issues
- Include steps to reproduce and environment details.
- Attach screenshots or logs when applicable.
